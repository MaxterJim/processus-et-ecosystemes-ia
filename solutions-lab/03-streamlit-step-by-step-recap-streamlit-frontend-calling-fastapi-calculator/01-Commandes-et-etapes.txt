├── streamlit/

├── main.py                     <- backend FastAPI (4 endpoints calculatrice)
├── app.py                      <- front Streamlit qui appelle le backend



==> Étape 3 - Il faut se placer dans la racine du projet  (demo2)

- Ouvrir le terminal 
- Exécutez la commandea suivante pour lancer l'ensemble de l'architecture :

Commande #1 : docker compose up -d --build
Commande #2 : docker compose ps

ou

Commande #1 : docker compose build
Commande #2 : docker compose up -d fastapi streamlit
Commande #3 : docker compose ps
